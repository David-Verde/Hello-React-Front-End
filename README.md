Getting Started
To get a local copy up and running follow these simple example steps.

Prerequisites
NPM 8.19.1 or higher
Modern web browser
Setup
git clone 
cd /hello_react_rails_frontend/
Install
npm install
npm run build
Usage
npm start
Then on a browser you can navigate to localhost.
